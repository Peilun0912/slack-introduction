Slack 的核心功能

  • 依照不同的需求建立公開或私人的討論頻道
  • 一對一通訊
  • 檔案分享
  • 強大的搜尋功能
  • 整合多個服務
  • 賣萌的 Slackbot
  • 提供 API 擴充功能
  • 跨平台

通訊管道的差異

Channels

公開的頻道，任何人都能加入討論。

Private Groups

私人頻道，只有受邀的人才能加入。適用於小型專案或是需要討論敏感資訊的時候。

Direct Messages

一對一聊天用。

檔案分享

可以分享圖片、PDF、試算表、doc 檔等文件。

可以分享 Google Drive、Dropbox 等雲端服務的文件，當這些文件更新時 Slack 也會跟著更新。

搜尋功能

  • 歷史訊息
  • 分享過的檔案名稱
  • 分享過的檔案的內文
  • 支援搜尋語法
